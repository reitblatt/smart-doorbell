# Smart Video Doorbell

Matter 1.5 compliant video doorbell with RTSP streaming for Home Assistant/Frigate integration.

## Features
- Matter 1.5 video doorbell device type
- RTSP streaming via MediaMTX
- Two-way audio with echo cancellation
- Raspberry Pi CM4 / Pi 4 compatible
